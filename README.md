# 📚 Anotações Técnicas

Minha base pessoal de anotações sobre tecnologia, reunindo conceitos, comandos, configurações, tutoriais e experiências adquiridas durante os estudos e no uso diário.

O objetivo deste repositório é servir como uma fonte de consulta rápida e um histórico de aprendizado, centralizando informações que normalmente ficariam espalhadas entre documentos, favoritos e anotações.

## 📂 Conteúdo

Os assuntos abordados incluem, entre outros:

- 🐧 Linux

- 🐳 Docker e Podman

- 🌿 Git e GitHub

- 🏠 Gitea

- 💻 VS Code

- 🌐 Redes

- 🐍 Python

- 📜 Shell Script

- ⚙️ DevOps

- 🔒 Segurança

- 🛠️ Troubleshooting

## 🎯 Objetivo

- Registrar conhecimentos adquiridos durante os estudos;

- Documentar configurações e procedimentos utilizados no dia a dia;

- Criar uma referência rápida para consultas futuras;

- Organizar conteúdos técnicos em um único lugar.

> **Observação:** As anotações refletem meu processo de aprendizado. O conteúdo pode ser atualizado, reorganizado ou expandido conforme novos conhecimentos são adquiridos.
> 
> As anotações deste repositório são produzidas durante meus estudos e revisadas sempre que necessário. Em alguns casos, utilizo ferramentas de IA como apoio para revisar explicações, validar informações e melhorar a organização do conteúdo. Todo o material é posteriormente conferido antes de ser incorporado ao repositório.
